Live URL: https://checkpoint-ruddy.vercel.app/
Github Repo: https://github.com/nathanrsxtn/checkpoint
Local Setup: git clone —> install dependencies (pnpm install) —> create .env file in directory following .env example format —>  pnpm run dev —> copy link into browser
Tech Stack List:
Frontend: React, Vite, React Router, Tailwind CSS, React Hot Toast
Backend: Node.js, Express
Database: MongoDB, Mongoose
Authentication: JSON Web Token, bcryptjs
Version Control & Development: git, github, pnpm, VScode
Deployment: Vercel, Render What we learned: Throughout developing the Checkpoint app, we learned how to build a full stack web application with React, Express, MongoDB, and Vercel/Render. We learned to use APIs and Loaders to communicate between front and backend, we learned how to use JS and CSS files to implement features and style them. We also learned how to authenticate using JWT and hashing, as well as managing, storing, and updated data using MongoDB and Mongoose. We also learned how to work as a team, keeping frequent communication about potential meetings, and features we were working on, as updates to the application. We also learned proper presentation habits through multiple milestone presentations as well as a final project presentation.
